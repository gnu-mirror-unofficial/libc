Undocumented functions in GNU libc

The following functions which are available in GNU libc are currently not documented in the GNU libc manual. If you have some time to spend consider writing some of the missing documentations. Texinfo is easy to use and if you don't like this, plain text is equally welcome (I'll do the rest).

The list might include some internal functions which need not be documented. I haven't compiled the list by hand, it is generated. So, it is in any case best to talk to me before starting to write. For one, I can make sure the function really needs to be documented and second, I can make sure no two people are working on the same function.
